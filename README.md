# My ROUTINE

自分専用の無骨でカッコいいルーチンワーク管理・分析Webアプリケーション

## 🎯 概要

「My ROUTINE」は、男性心をくすぐるハッカーライクで無骨なデザインの、個人用ルーチンワーク管理アプリです。
赤と黒を基調としたクールなUIで、日々のタスクを管理し、パフォーマンスを分析できます。

## ✨ 主な機能

### HOMEページ
- **日付表示**: 大きく見やすい日付表示 (YYYY/MM/DD形式)
- **モチベーショナル名言**: 偉人の名言をコードブロック風に表示（英語+日本語訳）
- **デイリールーチンリスト**: 
  - チェックボックス形式でタスクを管理
  - チェックすると「Done!」アニメーションとともにタスクが消える
  - 完了したタスクは自動的に記録される

### ANALYSTページ
- **パフォーマンス分析**:
  - Weekly/Yearlyの期間切り替え
  - 完了数の推移グラフ（赤いライン）
  - 直近1週間の達成率表示
- **ルーチンマスター管理**:
  - ドラッグ&ドロップで優先順位変更
  - 新規ルーチン追加
  - 既存ルーチン編集
  - ルーチン削除

## 🎨 デザインコンセプト

- **カラーパレット**:
  - 背景: 完全な黒 (#000000)
  - テキスト: 白 (#FFFFFF)
  - アクセント: 赤のグラデーション（左から右へフェードアウト）
- **タイポグラフィ**:
  - 一般テキスト: Sans-serif (Modern & Clean)
  - 名言部分: Monospace (JetBrains Mono)
- **UI言語**:
  - ラベル: Basic English (HOME, Done, etc.)
  - 分析: 日本語 (見やすさ重視)
  - ユーザーコンテンツ: 日本語

## 🚀 使い方

### ローカル環境での起動

1. **ファイルを開く**
   ```bash
   # ブラウザでindex.htmlを直接開く
   open index.html
   ```
   または、ブラウザでファイルをドラッグ&ドロップ

2. **簡易サーバーで起動（推奨）**
   ```bash
   # Pythonを使用する場合
   python3 -m http.server 8000
   
   # Node.jsを使用する場合
   npx http-server -p 8000
   ```
   ブラウザで `http://localhost:8000` にアクセス

### 基本操作

#### HOMEページ
1. 毎日午前0時にタスクリストがリセットされます
2. チェックボックスをクリックしてタスクを完了
3. 完了すると「Done!」アニメーションが表示され、タスクが消えます

#### ANALYSTページ
1. **グラフ表示**:
   - 「Weekly」ボタン: 直近7日間のデータ表示
   - 「Yearly」ボタン: 直近365日間のデータ表示
2. **達成率**: 直近1週間の達成率がパーセンテージで表示されます
3. **ルーチン管理**:
   - 新しいルーチンを入力して「追加」ボタンをクリック
   - カードの左側「⋮⋮」をドラッグして順番を変更
   - 「編集」ボタンで既存のルーチン名を変更
   - 「削除」ボタンでルーチンを削除

## 💾 データ保存

- **保存先**: ブラウザのLocalStorage
- **保存内容**:
  - ルーチンマスターデータ
  - 日次完了ログ
- **注意**: ブラウザのキャッシュをクリアするとデータが消えます

## 📦 技術スタック

- **フレームワーク**: バニラJavaScript (静的HTML)
- **スタイリング**: Tailwind CSS (CDN)
- **グラフ**: Chart.js
- **フォント**: JetBrains Mono (Google Fonts)
- **データ保存**: LocalStorage

## 🔧 カスタマイズ

### 名言の追加・編集
`index.html` 内の `QUOTES` 配列を編集してください：

```javascript
const QUOTES = [
    {
        text: "英語の名言",
        translation: "日本語訳",
        author: "著者名"
    },
    // 追加...
];
```

### モックデータの変更
`initializeMockData()` 関数内のデータを編集してください。
